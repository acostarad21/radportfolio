Street Sprinter
Street Sprinter

Street Sprinter is a browser-based endless runner game that takes inspiration from Chrome’s offline dinosaur game, but with added modern twists and a pixel-art visual style. Developed using Web Forms and simple file handling with Notepad as its data storage, it features multiple playable characters, progressively challenging gameplay, and a high score leaderboard. With vibrant, retro-inspired graphics and pop culture-themed sprites, the game offers responsive controls, animated obstacles, and limited lives, delivering a nostalgic yet refreshed arcade experience.

Spate
Spate E-Commerce Platform

Spate is a lightweight eCommerce website developed with ASP.NET for the backend and Microsoft Access as its database. It features a minimalist design that prioritizes user experience and easy navigation. The platform includes essential online shopping functionalities such as product display, cart handling, and a secure checkout process. By utilizing ASP.NET’s powerful server- side processing and MS Access for managing product details, user information, and orders, Spate showcases how simple technologies can support a fully functional web application. Its clean interface delivers quick loading times and a seamless experience across devices, making it a practical solution for small businesses and startups seeking a straightforward online store.

SafeStep
SafeStep

SafeStep is our capstone project, a mobile-based assistive navigation system designed for visually impaired individuals. Titled GPS-Enabled Obstacle Detection and Navigation System with Multi-Sensory Feedback for the Visually Impaired, the project combines various technologies to offer a more effective and user-friendly solution compared to traditional single-feedback tools. It utilizes the smartphone’s camera for real-time object detection, GPS for location tracking, and delivers feedback through both vibrations and voice prompts. To enhance real-time detection, we incorporated IoT components such as ultrasonic sensors and the ESP32-CAM module for basic image processing. This integrated system helps identify nearby obstacles and provides intuitive alerts, allowing users to move confidently through different environments. SafeStep is built to be practical, affordable, and scalable, aiming to improve mobility, reduce reliance on others, and foster greater inclusivity.
